JavaScript y el Navegador

Cuando dos o mas computadoras se enlazadan de manera que se puedan compartir informacion una a la otra se forma una red (network). Esto se logro desde los años 50, pero en los años 80 se creo la tecnologia que hacia posible una red a nivel mundial, a esta se le llamo internet.

Para una comunicacion efectiva es necesario que las computadoras conectadas interpreten la informacion de la misma manera, para esto es necesario un protocolo de red. Existen protocolos de red para diferentes cosas como envio y recepcion de correos electronicos, para compartir datos o para controlar computadoras infectadas.

Existe tambien el HyperText Transfer Protocol (HTTP) que es de los mas usados, sirve para compartir recursos nombrados como paginas web o imagenes, y cuenta con muchas reglas para transferencia de datos.

Mas dominante aun es el Transmission Control Protocol (TCP) ya que todos los dispositivos conectados a internet lo "hablan" y la mayoria estan directamente construidos sobre el. Esta funciona en base a puertos, cada puerto espera un tipo de transmision especifica, por ejemplo el puerto 25 es el que espera la recepcion de correos electronicos.

La world wide web (www) es un conjunto de protocolos que nos permite visitar paginas web en un navegador. Para ser parte de ella solo debes conectarte a internet y escuchar el puerto 80 con el protocolo HTTP para poder intercambiar documentos con otras computadoras.

Cada documento en internet se nombra con un URL (Uniform Resourse Locator) y cada maquina conectada a internet cuenta con una direccion IP, que es un numero que se utiliza para mandarse mensajes entre maquinas

HTML que abrevia HyperText Markup Language, es un formato de documento utilizado por las paginas de internet.Contiene textos, y etiquetas que les dan escritura a estos, describiendo consas como links, parrafos y encabezados.

Se puede incluir JavaScript en un docuemnto HTML mediante la etiqueta script
